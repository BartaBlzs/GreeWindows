# GREE Air Conditioner Control

An application which enables the user to connect and control GREE air conditioners on the local network from Windows.

## About

This project is heavily based on the excellent work of [tomikaa87/gree-remote](https://github.com/tomikaa87/gree-remote). 

**⚠️ Work in Progress**: This application is far from complete and is actively being developed.

## Download

If you don't want to build it yourself, you can download the executable from the [releases page](https://github.com/BartaBlzs/GreeWindows/releases).
Follow the instructions there for installation.
